# 精英怪与奖励盒

Yetrealm 的精英系统与世界难度、邪念苹果、技能材料和奖励盒相互连接。源码中的世界难度默认自然精英刷新概率为 10%，`Evil Apple` 会永久提高自然精英刷新概率。

## 奖励盒操作

- 右键打开，随机获得数件战利品。
- 按住 `Ctrl + 右键` 可一次打开整组。
- 奖励盒按 Tier I / II / III 划分，常作为精英怪与高级战斗内容的随机奖励容器。

## 难度交互

- Easy 模式在食用邪念苹果前不会自然刷新三级精英。
- Vanilla / Custom 可通过解锁天数控制三级精英生成，Custom 可使用 101 禁用自定义三级精英解锁。
- Hardcore Cursed 开局启用三级精英，并叠加硬核规则。

!!! tip "整合包建议"
    专家包可以降低常规矿物获取、提高精英权重，并让怪物精华成为技能树中期瓶颈；休闲包则建议保留奖励盒高价值但降低自然精英压力。
