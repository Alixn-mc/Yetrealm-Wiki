# Astral Realm

**Astral Realm** 是 Yetrealm 的星界终局维度，维度 ID 为 `yetrealm:astral_realm`。它不是由普通门框进入，而是通过 `Astral Keystone` 将五层 Beacon 转化为 **Astral Beacon** 后，在信标上方生成大型星界传送门。

## 维度特征

| 属性 | 值 |
| --- | --- |
| 维度 ID | `yetrealm:astral_realm` |
| 维度类型 | `yetrealm:astral_type` |
| 固定时间 | `18000` |
| 高度 | `min_y=0`，`height=384`，`logical_height=384` |
| 群系 | `yetrealm:astral_void` |
| 床 | 不可用 |
| 重生锚 | 可用 |
| 天光 | 有 |
| 环境光 | `0.08` |
| 天空效果 | `yetrealm:astral_sky` |

## 星界目标

- 找到 Stargazer 相关地点，使用 `Star Abyss Lens` 与 `Stargazer Observation Core` 互动。
- 击败 Stargazer，取得 `Star Abyss Core`。
- 解锁 `Cosmic Cube`、高阶星界材料与 Plasma 终局装备路线。
- 通过星界黑洞返回主世界；对应进度为“黑洞归来”。

## 星界敌人

Astral Worm、Photon Watcher、Starflare Sentinel 与 Stargazer 构成星界战斗梯度。它们常掉落 `Starlight Prism Crystal`、`Sun Sword`、`Sun Shield` 或 `Star Abyss Core` 等关键物品。

!!! warning "进入前检查"
    Astral Realm 是高压终局区域。建议至少完成 Primordial Alloy 阶段，并准备高机动背部饰品、远程输出、抗性/回复、Curios 护符与足够食物。
