# 远古宇宙资源路线

## 目标顺序

1. **建立传送门据点**：远古之门附近先封闭、照明、设置箱子和重生锚。
2. **陨铁时代**：寻找 Meteorite / Meteorite Ore，制作 Meteorite 工具与防具。
3. **青铜时代**：收集 Copper Tin Concentrate、Ancient Metal 和对应升级模板。
4. **深渊资源**：寻找 Abyss Treasure Map，探索深渊，取得 Abyssal Cell、Abyssal Razorfang、Abyss Crystal。
5. **火山资源**：进入 Ashen Caldera/火山区域，击杀 Ash Beast 或搜集 Ember Crystal。
6. **寒冷 Boss**：用 Ice Queen Sigil 召唤 Ice Queen，取得 Frozen Core。
7. **巨龙目标**：击败 Ancient Dragon，准备 Primordial Alloy。

## 装备推荐

- 初入：Night Vision Goggles、Rabbit Foot Boots、Magnet Ring、Simple Backpack。
- 深渊：Abyss Diving Goggles、Deep Abyss Trident、Abyss Rending Claw。
- 火山：Fire Guard / Ember Guard Ring、Meteorite/Ancient Metal 护甲。
- Boss：Frost Aurora Core、Dragon Ring、Warding Amulet、Resurrection Amulet。

## 风险控制

Cursed 或 Hardcore Cursed 下，远古宇宙常规怪物也会获得额外护甲和撕裂能力。不要把 Boss 资源路线当作纯采矿路线；每次远征应携带返程、治疗和备份装备。
