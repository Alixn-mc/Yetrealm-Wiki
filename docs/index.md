## Recommended reading order

1. Install Minecraft 1.21.1, NeoForge 21.1.x, Curios, Apothic Attributes, and GeckoLib.
2. Choose a world mode before starting a long-term world or server.
3. Follow the Ancient City -> Ancient Universe -> Astral Realm progression route.
4. Use the boss and data pages to plan gear, Curios, loot, and recipes.

!!! tip "One-line player route"
    Ancient City portal + Ancient Soul -> Ancient Universe -> Meteorite / Ancient Metal -> Ice Queen -> Ancient Dragon -> Primordial Alloy -> Astral Keystone + five-layer Beacon -> Astral Realm -> Stargazer -> Star Abyss Core -> Cosmic Cube / Plasma endgame.
