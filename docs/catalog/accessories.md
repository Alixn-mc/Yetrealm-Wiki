## 重要饰品说明

- **Abyss Rending Claw**：同时提供盔甲撕裂、盔甲穿透、保护撕裂、保护穿透和暴击率，是穿透向输出核心。
- **Warding Amulet**：专门针对 Reaper，降低其对玩家伤害并提高玩家对 Reaper 伤害。
- **Divinity**：正面效果时长变四倍，负面效果变四分之一，并能反转一次致死伤害后随机传送。
- **Frost Aurora Core**：提供冰霜伤害，并在短时间承受高额伤害后释放冰属性爆发。
- **Meteorite Wings / Starflow Wings / Featherlight Cloak**：背部机动类饰品，适合星界与高空门环境。
