# Blocks

This page is the English block catalog entry. Keep block IDs visible so players, server owners, and datapack authors can search logs and override resources.

## Important Block Groups

| Group | Examples | Use |
| --- | --- | --- |
| Portal blocks | `ancient_portal`, `astral_portal`, `black_hole_portal` | Dimension and route progression |
| Ancient materials | `ancient_rock`, `worldroot_log`, `worldroot_leaves` | Ancient Universe terrain and resources |
| Abyss blocks | `abyss_stone`, `abyss_tendril`, `abyss_crystal_cluster` | Abyssal Ocean exploration |
| Progression blocks | `meteorite_block`, `primordial_alloy_block`, `plasma_block` | Material storage and crafting identity |
| Utility blocks | `cosmic_cube`, `astral_beacon`, `stargazer_observation_core` | Late-game systems and summoning |

For generated item and block IDs, use the source assets under `docs/assets/yetrealm/textures/`.
