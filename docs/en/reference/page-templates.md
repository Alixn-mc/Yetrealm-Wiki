# Page Templates

## Feature Page

```markdown
# Page Title

Short player-facing summary.

## What it does

## How to get it

## Related pages
```

## Data Page

```markdown
# Page Title

Generated or audited data source.

| ID | Name | Notes |
| --- | --- | --- |
```

## Writing Rules

- Keep player instructions concrete.
- Preserve IDs in tables.
- Link route pages before giant reference tables.
- Do not invent behavior that is not in source, configs, or release notes.
