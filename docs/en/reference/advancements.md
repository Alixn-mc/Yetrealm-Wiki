# Advancements

Yetrealm advancements describe the main route and are useful for both player guidance and wiki maintenance.

## Key Route

| Area | Advancement purpose |
| --- | --- |
| Root | Introduces the Ancient Star Road route |
| Curios | Teaches early accessory use |
| Ancient Universe | Marks entry into the first major dimension |
| Bosses | Tracks Ice Queen, Ancient Dragon, Reaper, and Stargazer milestones |
| Astral Realm | Tracks Astral Beacon, Stargazer, Star Abyss Core, and endgame progress |

When updating this page, compare `src/main/resources/data/yetrealm/advancement/` with the visible language strings in `en_us.json`.
