# Config

Yetrealm configuration matters most for servers and modpacks. Document choices before opening a public world.

## High-Impact Areas

| Config area | Why it matters |
| --- | --- |
| World mode | Changes survival difficulty, monster scaling, hunger, luck, and hardcore rules |
| Curios slots | Controls how many accessory slots players receive |
| Cosmic Cube safety | Affects multiplayer storage risk and recovery expectations |
| Food and hunger | Changes long-term sustain and food variety incentives |
| Elite mobs | Changes spawn pressure, rewards, and expert-pack difficulty |

## Release Recommendation

- Normal modpacks: start with Challenge.
- Expert modpacks: test Cursed or Hardcore Cursed before public release.
- Public servers: document any Custom values and keep a backup before changing them.
