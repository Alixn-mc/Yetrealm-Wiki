# Cosmic Cube

Cosmic Cube is a late-game storage and utility system. It belongs in both player progression and server safety documentation because storage behavior can matter on multiplayer worlds.

## Player Use

- Long-term storage and item organization.
- Late-game progression target tied to Astral and Star Abyss materials.
- Works best after the player has stable Ancient Universe and Astral Realm access.

## Server Notes

- Decide whether storage and access rules need server-specific limits.
- Document any modpack restrictions before launch.
- Re-test after version updates that touch menus, saved data, or item storage.
