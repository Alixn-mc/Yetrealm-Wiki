# Systems

This page groups Yetrealm's supporting systems so the CurseForge description can link to gameplay features without overloading the overview.

## System Summary

| System | Player-facing use |
| --- | --- |
| Skill Tree | Long-term stat planning through XP and material costs. |
| World Mode | Configurable world difficulty and related progression tuning. |
| Cosmic Cube | Late-game storage and category/sorting workflow. |
| Food diminishing returns | Food variety and sustain balancing for long survival worlds. |
| Elite Mobs | Higher-pressure enemy variants with levels, effects, gear, and mounts. |
