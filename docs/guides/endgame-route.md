# 星界与终局路线

## 进入前置

- 完成 Primordial Alloy 阶段。
- 制作 `Astral Keystone`。
- 在主世界搭建五层 Beacon，并确保上方有足够空间。
- 右键信标，转化为 Astral Beacon，进入上方星界门。

## Astral Realm 推进

1. 探索 Astral Void，收集 `Astral Energy` 与 `Starlight Prism Crystal`。
2. 寻找 Stargazer Observation Core。
3. 使用 `Star Abyss Lens` 唤醒 Stargazer。
4. 击败 Stargazer，取得 `Star Abyss Core`。
5. 制作/获取 Cosmic Cube，进入终局存储与 Plasma 装备路线。
6. 通过星界黑洞返回主世界，完成返航进度。

## 终局构筑方向

- **Plasma 装备**：终局采掘等级与不可破坏装备。
- **Cosmic Cube**：长期存储与共享空间。
- **高阶饰品**：Dragon Soul、Frost Aurora Core、Divinity、Starflow Wings 等提供极强防御/机动/容错。
- **技能树补完**：将核心输出、生存和探索节点升满，补齐命中率、护甲穿透、最大伤害吸收和机动属性。
