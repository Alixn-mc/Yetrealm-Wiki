# Ancient Universe

**Ancient Universe** 是 Yetrealm 的第一个主线维度，维度 ID 为 `yetrealm:ancient_universe`。入口来自主世界 Ancient City 中央门形结构：在强化深板岩上使用 `Ancient Soul` 后，模组会建立主世界与远古宇宙之间的成对传送门。

## 维度特征

| 属性 | 值 |
| --- | --- |
| 维度 ID | `yetrealm:ancient_universe` |
| 维度类型 | `yetrealm:ancient_universe_type` |
| 高度 | `min_y=-64`，`height=512`，`logical_height=512` |
| 床 | 不可用 |
| 重生锚 | 可用 |
| 天光 | 有 |
| 天空效果 | `yetrealm:ancient_sky` |
| 地形源 | 自定义 `yetrealm:volcanic_caldera` biome source + noise generator |

## 资源和目标

- **陨铁线**：`Meteorite` 与 `Meteorite Ingot` 是第一套远古维度装备材料。
- **青铜线**：`Copper Tin Concentrate` 与 `Ancient Metal` 推动 Ancient Metal 装备与升级模板。
- **深渊线**：深渊遗迹、深渊植物、Abyss Crystal、Abyssal Cell 和 Abyssal Razorfang 连接水下战斗与 Deep Abyss Trident。
- **火山线**：Ashen Caldera 与 Ember Crystal 是 Astral Keystone 及若干终局物品的重要材料来源。
- **Boss 线**：Ice Queen、Ancient Dragon、Reaper 等远古宇宙 Boss/强敌决定后续材料和饰品能力。

!!! tip "建议落脚点"
    首次进入后优先建立安全传送门据点，准备水下呼吸/夜视/高护甲，再向深渊、火山和寒冷远古群系推进。
